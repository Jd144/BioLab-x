# BioLabX

BioLabX is a clean, responsive React + Vite + Tailwind CSS frontend for an AI-powered virtual biotechnology lab platform. It uses dummy data only and does not include a backend yet.

## Features

- Home page for the BioLabX platform
- Student Dashboard with learning progress and AI guidance
- Experiments catalog with simulated biotechnology activities
- Gel Electrophoresis Simulator interface
- Teacher Dashboard with class insights
- Inventory page for lab materials and equipment
- Professional academic light theme
- Mobile responsive navigation and layouts

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React icons

## Setup Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/Jd144/BioLab-x.git
   cd BioLab-x
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Build for production:

   ```bash
   npm run build
   ```

## Notes

- This project is frontend-only.
- All content is dummy data.
- Backend, authentication, persistent storage, and real AI integrations can be added later.
